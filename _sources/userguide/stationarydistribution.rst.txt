Final stationary distribution
=============================

The stationary distribution at the final iteration. This data set is saved as a CSV file with name ``Final_stationary_distribution_CURRENT_INSTANCE_NAME`` if ``SAVE_OUTPUT=True`` in a folder named according to ``INSTANCE_OUTPUT_DIRECTORY``.
