Objective plot#

If PLOT_OBJECTIVE=True, a plot of the objective value at each iteration is created. If SAVE_OUTPUT=True, the plot is saved as a pdf file with name CURRENT_INSTANCE_NAME_objective_function in a folder named according to INSTANCE_OUTPUT_DIRECTORY.
